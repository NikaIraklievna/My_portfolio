**Проект 2**: Когортный анализ жизни студентов онлайн школы с помощью SQL

Аналитическая задача: разобраться почему студенты, начавшие занятия в апреле 2016 проходят меньше уроков

Этапы моего решения: 
1. Разбила студентов на когорты по месяцу успешного прохождения 1 урока
2. Вычислила время жизни каждой когорты накопительно
3. Посчитала количество студентов в когорте и количество активных студентов в разные месяцы жизни
4. Посчитала метрики :
   - долю активных студентов по месяцам жизни
   - среднее количество занятий на активного студента
5. Визуализацию, выводы и код можно посмотреть в презентации: [Когортный анализ жизни студентов, пришедших в первом полугодии 2016 года (https://docs.google.com/presentation/d/1tvOTfHM9xztjmNuhVCjao6FyRkjCjRccalFz8_fSahM/edit?usp=sharing)

**Проект 3**: Моделирование изменения балансов студентов в онлайн школе

Аналитическая задача: Сформировать баланс уроков каждого студента за каждый "прожитый" им день в 2016 году

Этапы моего решения:
1. Собрала календари для каждого студента от даты его первой оплаты до конца 2016 года
2. Добавила в календари изменения баланса связанные с успешными транзакциями
3. Добавила в календари изменения баланса связанные с прохождением уроков
4. Сформировала единый календарь с ежедневным изменением баланса каждого студента
5. Визуализацию, выводы и код можно посмотреть в презентации: [Изменение баланса уроков студентов онлайн школы на протяжении 2016 года](https://docs.google.com/presentation/d/1SHdVP0RgwjkO6xa4gV3uHtnUZHn_LU7joAXpFqxTz9w/edit?usp=sharing)
